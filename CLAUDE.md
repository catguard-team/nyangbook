# nyangbook/CLAUDE.md

> 라즈베리파이 기반 자경단 보급 디바이스. AGENTS.md를 먼저 읽으세요.

## 이 레포의 역할

라즈베리파이 OS 커스터마이징, 부팅 스크립트, 자경단 콘텐츠 런처 등 **냥북** 디바이스의 모든 것.

## 타겟 하드웨어

- **냥북 미니**: 라즈베리파이 5 (4GB) + 7인치 디스플레이 (또는 HDMI TV)
- **냥북 라이트**: 리퍼 크롬북 또는 폐기 노트북
- **할매폰**: 안드로이드 태블릿 (별도 트랙)

## 작업 영역

- `os/` — Raspberry Pi OS 커스텀 (이미지, 스크립트)
- `launcher/` — 자경단 콘텐츠 런처 (큰 글씨, 어린이 친화 UI)
- `content/` — 모비딕 콘텐츠 통합 (사주 어린이판, 식품, 법률 등)
- `boot/` — 부팅 시 행동강냥 표시
- `docs/` — 보급 매뉴얼, 셋업 가이드

## 작업 원칙

1. **OLPC 실패 교훈 반영**: 자체 OS 만들지 말 것. 표준 환경 위에 콘텐츠 레이어만.
2. **유지보수 우선**: 신기능보다 안정성. 자경단원이 원격 지원 가능한 구조.
3. **부모 동의 가능 콘텐츠**: 어린이용은 부모/보호자가 봐도 안심할 수 있어야.

## 자주 쓰는 환경

- Raspberry Pi OS Bookworm
- Python 3.11+
- Bash / Shell 스크립트
- Ansible (배포 자동화 가능)
